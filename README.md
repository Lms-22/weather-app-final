# 나만의 날씨 예보 웹사이트 (기말 과제)

## 1. 프로젝트 소개
HTML, CSS, JavaScript를 활용하여 만든 날씨 예보 사이트입니다.
OpenWeatherMap API를 사용하여 실시간 날씨와 5일 예보를 보여줍니다.

## 2. 주요 기능
- 도시 검색 및 최근 검색어 저장
- 현재 날씨 및 5일 예보 카드 표시
- 섭씨/화씨 단위 변환
- (본인이 추가한 확장 기능, 예: 현재 위치 기반 날씨)

## 3. 기술 스택
- HTML5, CSS3
- JavaScript (Vanilla JS)
- OpenWeatherMap API
- Vercel Deployment
